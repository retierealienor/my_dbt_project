{% docs mart_customers_by_store %}

This model provides a summary of user activity, focusing on their order history and purchasing behavior. It aggregates key metrics for each user, such as:

- **Nb Customers**: The total amount of registered customers.
- **Nb Store**: Number of stores.
- **Nb Staff**: The total amount of staff.

This allows for analysis of performance by account manager and region.
